# Hospital Readmission Analysis (SQL Project)

## Overview
This project analyzes hospital patient data to identify patterns associated with patient readmission. The goal is to explore how factors such as diagnosis, age, and length of stay relate to readmission risk using SQL-based data analysis.

## Objective
To practice SQL querying and perform exploratory data analysis on healthcare data by:
- Identifying diagnoses with the highest readmission rates
- Analyzing how age groups impact readmission likelihood
- Exploring the relationship between length of stay and readmission risk

## Dataset
A synthetic dataset of hospital patients containing the following variables:
- Patient ID
- Age
- Gender
- Diagnosis
- Length of hospital stay
- Readmission status (True/False)

## Tools Used
- PostgreSQL
- pgAdmin

## Key Analyses Performed

### 1. Readmission Rate by Diagnosis
Evaluates which medical conditions are associated with higher readmission rates.

### 2. Age Group Analysis
Groups patients into age ranges to determine how age impacts readmission probability.

### 3. Length of Stay vs Readmission
Analyzes whether longer hospital stays correlate with higher readmission risk.

## Key Insights
- Certain diagnoses (e.g., pneumonia and heart failure) show higher readmission rates.
- Older age groups demonstrate increased likelihood of readmission.
- Longer hospital stays are generally associated with higher readmission probability.

## Skills Demonstrated
- SQL querying (SELECT, GROUP BY, ORDER BY)
- Conditional logic using CASE statements
- Aggregate functions (COUNT, SUM, AVG)
- Exploratory data analysis
- Data interpretation for healthcare insights
